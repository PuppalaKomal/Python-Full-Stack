SQL Commands/ SQL sublanguages
    1. DDL: Data Definitation Language
    2. DML: Data Manuplation Language
    3. DQL: Data Query Language
    4. DCL: Data Control Language
    5. TCL: Transaction Control Language 

    1. DDL:(OPERATIONS ON STRUCTURE)(auto-commited commands which means rollback is not possible)
        1. CREATE: (create the table)
        2. ALTER: (modify the table structure)
        3. DROP: (delete the table)
        4. TRUNCATE: (delete the entire table records)
        5. RENAME: (renames the table)
    2.DML:(RELATED TO TABLE)(not auto-commited commands ,rollback is possible)
        1. INSERT: (inserts data into fields or records)
        2. UPDATE: (updates the existing data)
        3. DELETE: (deletes the data based on the condition)
    3.DQL:()
        1.SELECT: (to get a particular data)
